# Kubernetes — Run the Docs

Learn Kubernetes from the official docs, explained in short videos.

## About This Series

Each episode covers one Kubernetes concept, straight from the [official documentation](https://kubernetes.io/docs/).

- **Audience:** Developers new to Kubernetes, DevOps engineers
- **Format:** 2–5 minute videos with visual explanations
- **Docs source:** https://kubernetes.io/docs/
- **Code examples:** https://github.com/run-the-docs/kubernetes
