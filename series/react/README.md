# React — Run the Docs

Learn React from the official documentation, explained in short videos.

## About This Series

Each episode covers one React concept, straight from the [official documentation](https://react.dev/learn).

- **Audience:** Developers new to React, frontend engineers learning the framework
- **Format:** 60–90 second videos with visual explanations and code examples
- **Docs source:** https://react.dev/learn
- **Code examples:** https://github.com/run-the-docs/react
